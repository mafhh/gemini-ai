# InnovTeams AI POCs - InnovTeams

Welcome to the **InnovTeams AI Proof-of-Concepts (POCs)** repository.  
These POCs demonstrate AI capabilities integrated with **InnovTeams**, our customer experience platform.

---

## Overview

This repository contains two main AI POCs:

1. **Sentiment Analysis**
   - Analyze customer messages in real-time to detect **sentiment** (positive, neutral, negative) and **emotion** (angry, frustrated, happy, calm, etc.).
2. **Knowledge Base Reply Suggestions**
   - Suggest **context-aware replies** based on a structured **knowledge base (KB)**, helping agents respond faster and more accurately.

These POCs use **Google Gemini 2.5 Flash** for AI inference.

---

## Features

- **Real-time Sentiment Analysis:** Understand customer mood and prioritize responses.
- **KB-driven AI Suggestions:** Provide agents with accurate replies drawn from FAQs and documentation.
- **Terminal-based POCs:** Easy to run locally for testing and demonstration.
- **Safe handling of API keys:** Uses `.env` for sensitive credentials.
