# ResolveCX AI – Sentiment & Reply Suggestion POC

Built by **InnovTeams** as a proof of concept for the ResolveCX platform.

## Features
- Multi-tenant SaaS architecture
- One tenant → multiple customers
- Concurrent, isolated AI requests
- Sentiment analysis
- AI-powered reply suggestions
- Gemini API integration

## Architecture
- Node.js (ES Modules)
- Tenant-scoped knowledge bases
- Stateless AI requests
- Secure data isolation
